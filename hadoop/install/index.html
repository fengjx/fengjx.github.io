hadoop入门-伪分布式安装

快速入门hadoop，仅搭建伪分布式，先掌握hadoop的基本使用，先掌握脉络再研究其原理。

参考：

  • https://hadoop.apache.org/docs/stable/hadoop-project-dist/hadoop-common/SingleCluster.html#Pseudo-Distributed_Operation

hadoop依赖java环境，需自行安装

下载、安装

官方下载地址：https://hadoop.apache.org/releases.html，选择适合的版本（这里选择2.10.0）

wget http://mirror.bit.edu.cn/apache/hadoop/common/hadoop-2.10.0/hadoop-2.10.0.tar.gz
tar -zxvf hadoop-2.10.0.tar.gz

# 包含目录
- bin
- etc
- include
- lib
- libexec
- LICENSE.txt
- logs
- NOTICE.txt
- README.txt
- sbin
- share

设置环境变量

# 根据实际情况修改环境变量配置文件
$ vim ~/bashrc

# 将下面内容追加到文件中
export HADOOP_HOME=${HOME}/opt/hadoop/hadoop-2.10.0
export PATH=:$PATH:$HADOOP_HOME/bin

# 测试hadoop是否安装成功
$ hadoop version

修改配置文件

etc/hadoop/core-site.xml

<configuration>
    <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:9000</value>
    </property>
</configuration>

etc/hadoop/hdfs-site.xml

<configuration>
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
</configuration>

设置免密码登录

# 如果已经生成过，可以跳过
$ ssh-keygen -t rsa -P '' -f ~/.ssh/id_rsa
$ cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
$ chmod 0600 ~/.ssh/authorized_keys

启动hdfs

  1. 格式化文件系统
$ bin/hdfs namenode -format
  1. 启动NameNode和DataNode
$ sbin/start-dfs.sh
  1. 通过浏览器访问

http://localhost:50070/

  1. 测试文件系统
# 创建目录
$ hdfs dfs -mkdir input
# 上传文件
$ hdfs dfs -put etc/hadoop/*.xml input

yarn单节点配置

  1. 修改follows配置文件

etc/hadoop/mapred-site.xml

<configuration>
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
    <property>
        <name>mapreduce.application.classpath</name>
        <value>$HADOOP_MAPRED_HOME/share/hadoop/mapreduce/*:$HADOOP_MAPRED_HOME/share/hadoop/mapreduce/lib/*</value>
    </property>
</configuration>

etc/hadoop/yarn-site.xml

<configuration>
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
    <property>
        <name>yarn.nodemanager.env-whitelist</name>
        <value>JAVA_HOME,HADOOP_COMMON_HOME,HADOOP_HDFS_HOME,HADOOP_CONF_DIR,CLASSPATH_PREPEND_DISTCACHE,HADOOP_YARN_HOME,HADOOP_MAPRED_HOME</value>
    </property>
</configuration>
  1. 启动ResourceManager和NodeManager
$ sbin/start-yarn.sh
  1. 通过浏览器访问ResourceManager

http://localhost:8088/

  1. 启动历史服务

可以看到mr执行历史

$ sh mr-jobhistory-daemon.sh start historyserver
